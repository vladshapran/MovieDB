# MovieDB
 
Приложение состоит с списка фильмов. При нажатии на секцию отображает информацию о фильме(Название, описание, дата релиза, картинка), при зажатии секции на 1 сек данный фильм попадает в список "Любимые", где отображено название фильма, при нажатии на секцию можно посмотреть какой у фильма жанр и его название. Так же возможно удалить фильм из любимого списка при зажатии на секцию в течении 1 сек.
